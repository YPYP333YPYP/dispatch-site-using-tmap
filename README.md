## 소개

Tmap API를 이용하여 Django 웹 프레임워크로 제작한 배차 웹 사이트

## 사용법
Center -> 물류 배송의 시작점이 되는 물류 창고 정보 입력
Zone -> 차량이 이동하는 코스의 권역 정보 입력
Vechicle -> 물류를 배달할 차량 정보 입력
Order -> 배송지 정보 입력

입력된 내용을 기반으로 Tmap API에서 자동 배차
